PHL Bike Theft REST API

Disclaimer: This is currently a prototype, developed in a rushed few hours at a hackathon. There may be bugs or inaccuracies. The team that developed it is planning to improve it in the coming months.

Info

This application provides a REST API around the existing bike theft data provided by the City of Philadelphia. It's designed to make it easier to query for data regarding bike theft locations. Instructions on how to construct a query are listed below (results are returned in a JSON array of GeoJSON Points).

Query Instructions

URL: http://biketheft.jeffsbox.net/search

Query Paramters (e.g. put these in the URL)

  • latitude=(latitude)
  • longitude=(longitude)
  • radius=(miles from latitude/longitude)
  • [optional] start_date=(YY-MM-DD)
  • [optional] end_date=(YY-MM-DD)
  • [optional, but requires next parameter too] start_hour=(0 through 23)
  • [optional] end_hour=(0 through 23)

If no time parameters are set, all the matching data available within the radius is returned

Example URL:

http://biketheft.jeffsbox.net/search?latitude=39.929081852&longitude=-75.1672029&radius=0.1
